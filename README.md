# R2Gen — Radiology Report Generator

A production-ready web interface for automated chest X-ray report generation. Upload an X-ray image and receive a structured radiology report — generated by a fine-tuned vision-language model in seconds.

🔗 **Live Demo:** [r2gen-app.vercel.app](https://r2gen-app.vercel.app)  
📄 **Model & Backend:** [master-project repo](https://github.com/ai-art-dev99/master-project)

---

## What it does

Most radiology AI research stops at the notebook. This project goes further — the model is served via a FastAPI backend on HuggingFace Spaces and consumed by a clean React frontend that any radiologist or developer can interact with directly.

Upload a chest X-ray → the model reads it → a structured clinical report comes back.

---

## Architecture

```
User uploads X-ray (React + Vite frontend)
        ↓
FastAPI backend on HuggingFace Spaces
        ↓
VisionEncoderDecoder: Swin Transformer + Cerebras-GPT
(fine-tuned on IU-Xray dataset, 4-bit NF4 quantised)
        ↓
Structured radiology report returned
```

## Getting Started

```bash
# Clone the repo
git clone https://github.com/ai-art-dev99/r2gen-app.git
cd r2gen-app

# Copy environment variables
cp .env.example .env
# Set VITE_API_URL to your backend endpoint

# Install dependencies
npm install

# Run locally
npm run dev
```

**Or with Docker:**
```bash
docker-compose up --build
```
